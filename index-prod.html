API endpoints

Esta documentação detalha os endpoints disponíveis na API Taks Manager

Autenticação

A autenticação é feita através de uma API Key. Ela deve ser transmitida em todas as requisições no header task_token , para testar use o valor 123456

Header

{
"task-token": 123456,
"Content-Type": "application/json"
}

Retorna uma lista com todas as tarefas cadastras por ordem de cadastro

GET   http://glaubertask.ddns.net/bdr_task/api/task/list
Response (Array of objects)
[
  {"id":1,"time_hour":1,"time_minute":30,"description":"Tarefa número 1"},
  {"id":2,"time_hour":0,"time_minute":15,"description":"Tarefa número 2"}
]

Retorna detalhes da tarefa filtrado por ID

GET   http://glaubertask.ddns.net/bdr_task/api/task/get/{{ID}}

Parameters

ID
Identificador único da tarefa
(Number)
Response (Object)
{"id":1,"time_hour":1,"time_minute":30,"description":"Tarefa número 1"}

Salva dados da nova tarefa realizada

POST   http://glaubertask.ddns.net/bdr_task/api/task/save

Parameters

time_hour
Quantidade de hora(s)
(Number)
time_minute
Quantidade de minutos(s)
(Number)
description
Descrição da tarefa realizada
(String)
Response (Object)
{ "type": "success", "message": "New task saved successfully!", "id": 1 }

Altera dados da tarefa filtrado por ID

PUT   http://glaubertask.ddns.net/bdr_task/api/task/edit/{{ID}}

Parameters

ID
Identificador único da tarefa
(Number)
Response (Object)
{ "type": "success", "message": "Task altered successfully!" }

Exlui dados da tarefa filtrado por ID

DELETE   http://glaubertask.ddns.net/bdr_task/api/task/delete/{{ID}}

Parameters

ID
Identificador único da tarefa
(Number)
Response (Object)
{ "type": "success", "message": "Task deleted successfully!" }
